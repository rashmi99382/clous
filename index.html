Simple AWS User Media Website

Create accounts, verify email, and upload media securely.

A clean starter project for a small user media platform using Amplify Hosting, Amazon Cognito, Amazon S3, and DynamoDB.

What users can do

Account, upload, dashboard.

Create account

Users sign up with name, email, and password. Cognito sends the verification code.

Reset password

Cognito handles forgot password emails and reset codes, so passwords never go into DynamoDB.

Upload media

Images and videos upload to a private user folder in S3 after login.

View dashboard

DynamoDB stores names, titles, descriptions, file paths, and upload dates.
